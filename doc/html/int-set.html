Library Module: int-set.xqm

URI: http://www.woerteler.de/xquery/modules/int-set
Description: Abstract interface for a set of integers.
Author: Leo Woerteler <leo@woerteler.de>
Version: 0.1

Functions

contains($set, $x)

Arguments:
$set item()* set to search in
$x xs:integer element to look for
Returns:
item()* true() if the element is contained in the set, false() otherwise
Description: Checks if the given element is contained in the given set.

empty()

Returns:
item()* the empty set
Description: Returns the empty set.

insert($set, $x)

Arguments:
$set item()* set to insert into
$x xs:integer element to insert
Returns:
item()* the set that contains $x in addition to all elements of $set
Description: Inserts the given integer into the given set and returns the resulting set.
