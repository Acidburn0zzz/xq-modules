Library Module: rng.xqm

URI: http://www.woerteler.de/xquery/modules/rng
Description: A simple random number generator implemented as a Linear Congruential Generator.
Author: Leo Woerteler <leo@woerteler.de>
Version: 0.1

Functions

random-ints($seed, $n)

Arguments:
$seed xs:integer the seed to start from
$n xs:integer number of random ints to produce
Returns:
xs:int* sequence
Annotations:
%public
Description: Generates $n random xs:ints, starting with the given seed.

with-random-ints($seed, $n, $start, $f)

Arguments:
$seed xs:integer seed for the random number generator
$n xs:integer number of random numbers to generate
$start item()* initial value for the fold
$f function(item()*, item()*) as item()* combining function
Returns:
item()* result of the fold
Annotations:
%public
Description: Folds over a sequence of $n randomly generated xs:ints, starting with the value $start and successively adding random ints by calling $f with the current value and the next int.
