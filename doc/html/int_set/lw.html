Functions

contains($set, $x)

Arguments:
$set item()* the Red-Black Tree
$x xs:integer the integer

item()* true() if the integer is contained in the tree, false() otherwise

empty()

Returns:
item()* empty set
Description: Empty set, which is the empty Red-Black Tree.

insert($set, $x)

Arguments:
$set item()* the Red-Black Tree
$x xs:integer the integer
Returns:
item()* tree where $x was inserted
